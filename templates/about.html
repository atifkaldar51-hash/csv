About

A clean, modern toolkit for instant CSV insights.

Project Overview

CSV Analytics Dashboard is a lightweight yet powerful web application that lets you upload any CSV file and immediately understand its shape, quality, and story. Whether you are exploring a fresh export from your CRM or sanity-checking a dataset before feeding it into a machine-learning pipeline, the dashboard gives you a clean preview, automatic statistics, and one-click Bar / Line / Pie charts — all wrapped in an Apple-inspired, glassmorphism-flavoured interface.

The project emphasises clarity over complexity: a single Flask backend, Pandas for data wrangling, Matplotlib for rendering, and Bootstrap 5 plus a small custom stylesheet for the front-end. No build step, no front-end framework lock-in — just clean, readable code you can extend in any direction.

Backend Stack

  • Flask — lightweight WSGI web framework for routing & APIs
  • Pandas — DataFrame-based data loading, preview & statistics
  • NumPy — numeric backbone for descriptive stats
  • Matplotlib — server-side chart rendering (Bar / Line / Pie)
  • Werkzeug — WSGI utilities and secure file handling

Frontend Stack

  • HTML5 — semantic, accessible page structure
  • CSS3 — custom Apple-inspired glassmorphism theme
  • JavaScript — drag & drop, AJAX chart generation
  • Bootstrap 5 — responsive grid & components
  • Bootstrap Icons — clean, modern icon set

Key Features

Drag & drop CSV upload with AJAX validation
First-10-rows preview with data types
Automatic summary cards (rows / columns / missing)
Numeric vs categorical column classification
Descriptive statistics table (count/mean/std/min/max)
Bar / Line / Pie chart generation with Matplotlib
Chart gallery with one-click PNG download
Graceful handling of empty / corrupted / wrong-format files

How the pipeline works

  1. Upload — your CSV is saved to static/uploads/ with a unique prefix and validated by Pandas.
  2. Parse — utils/csv_handler.py loads the DataFrame and computes the summary & preview.
  3. Render dashboard — the dashboard template displays cards, the preview table, and statistics.
  4. Generate charts — utils/chart_generator.py renders Matplotlib PNGs into static/charts/.
  5. Download — every chart is served as a high-resolution PNG via a dedicated download route.

Roadmap

  • Export analytics summary as PDF / Excel
  • Interactive Chart.js previews in addition to Matplotlib PNGs
  • User sessions with persistent upload history
  • Advanced charts: scatter, heatmap, histogram, boxplot
  • API token authentication for programmatic uploads
  • Light / Dark mode toggle
